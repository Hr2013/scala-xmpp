scala-xmpp is a scala based xmpp framework providing implementations of core xmpp stanza encoding/decoding and a framework for building external components. scala-xmpp can be used to build both server and client xmpp applications.  

scala-xmpp is similar to ejabbered's exmpp and openfire's tinder/whack xmpp libraries but is wriiten is native scala utilizing some of scala's excellent concurrency features and takes a different approach on stanza encoding/decoding.
